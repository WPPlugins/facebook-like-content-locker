With this powerful Content Locker you can lock your entire blog for a user, until he hits the Facebook Like Button.

== Description ==

You have various settings that you can edit in this powerful plugin:
-Set custom background color for widget
-Set custom color for transparent "blog" cover
-Set transparency for the widget
-Enter Facebook Admins for the page
-Set Facebook meta data (title, image, description etc...)


== Installation ==
1. Go to Plugins section from Wordpress Admin
2. Click on Add New Plugin and select "Upload"
3. Select the zip containing the plugin and press Upload
4. Activate the plugin and go to the Settings section to set your blog's details

== Frequently Asked Questions ==

= How can I add the Facebook Content Locker (Like Locker) on my page? =

You need to go to the configuration section of the plugin and include the tag [facebook] in the WYSIWYG from the appearance tab.

= Can I use this content locker without a Facebook Like button? =

Yes, you can add whatever content you want to your widget.
